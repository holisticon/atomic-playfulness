# Getting Started

Welcome to the documentation of the Atomic Playfulness design system.

## Installation

```bash
npm i @holisticon/hap-foundation
```

## Usage

To get access to the Atomic Playfulness design tokens, fonts, and css-class
elements, you can import its main CSS entrypoint in your application.

```js
import "@holisticon/hap-foundation/atomic-playfulness.css";
```
