# Color Tokens

Use semantic tokens (non-primitives) for UI work. Primitives are shown for
reference and design exploration but should not be used directly in components
or Tailwind classes.

## Semantic Colors (Use These)

These are the actual color tokens intended for UI usage. Each token maps to a
primitive but is stable for design.
