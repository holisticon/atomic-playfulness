# **Available Tailwind Class Names**

In the first few examples here we also show the CSS tokens that the classes are
derived from in order for you to understand the structure. The only important
thing are the actual class names. For these we also provide some examples of
usage with code snippets.

## **1. Spacing** (Margin, Padding, Gap, etc.)

- `none` → `var(--hap-spacing-none)`
- `xs` → `var(--hap-spacing-xs)`
- `sm` → `var(--hap-spacing-sm)`
- `md` → `var(--hap-spacing-md)`
- `mdlg` → `var(--hap-spacing-md-lg)`
- `lg` → `var(--hap-spacing-lg)`
- `xl` → `var(--hap-spacing-xl)`
- `2xl` → `var(--hap-spacing-2-xl)`

**Usage Example:**

```html
<div class="p-mdlg m-lg">Content</div>
```

---

## **2. Border Radius**

- `rounded-none` → `var(--hap-radius-none)`
- `rounded-minimal` → `var(--hap-radius-minimal)`
- `rounded` → `var(--hap-radius-rounded)`
- `rounded-large` → `var(--hap-radius-rounded-large)`
- `rounded-full` → `var(--hap-radius-full)`

**Usage Example:**

```html
<button class="rounded-lg p-md bg-fill-primary-light-default">Click Me</button>
```

- `bg-fill-brand-dark-default`

- `bg-fill-state-on`
